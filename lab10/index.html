Lab 10 - JavaScript for the Web

Challenge

The challenge of this lab was to be able to experiment with DOM manipulation through jQuery and JavaScript. DOM manipulation is an important part of web development since you can create web applications that update the data in a web page without having to refresh the page. This allows for a lot more designs and processes that can be done on a page without having to refresh it every time.
