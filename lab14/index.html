For this lab, I was getting errors in my console because of using "document.writeln". Using document.writeln with asynchronously loaded scripts just does not work at all and has the chance to overwrite the entire document. It's easer to use some DOM manipulation instead, so I made a function called addTextToBody that creates a paragraph element, sets its text content, and then appends it to the body. Adding this doesnt make a super drastic, insane change to the site or anything, it just gets rid of the errors in the console and cleans things up a little.
